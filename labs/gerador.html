📖 Como usar

1. Suba o laboratório

vagrant up

2. Acesse a máquina

vagrant ssh

3. Gere novos desafios

generate-ctf

4. Encontre os arquivos em

cd /cases/generated && ls -la

🎯 Tipos de puzzles gerados

  • base64.txt - Mensagem codificada em Base64
  • hash.txt - Hash MD5 para quebrar
  • protected.zip - ZIP com senha
  • image.jpg - Metadados EXIF escondendo a flag
  • strings.bin - Arquivo binário com texto oculto
  • xor.bin - Criptografia XOR simples
  • hex.txt - Mensagem em hexadecimal
  • binario.txt - Mensagem em binário
  • rot13.txt - Cifra ROT13
  • http.pcap - Captura de rede simulada
