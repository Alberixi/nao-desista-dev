> 📡 API DOCUMENTATION

Base URL: https://api.naodesista.dev/v1

🔐 Autenticação

POST /auth/login

Autentica um agente e retorna um token JWT.

{ "username": "agente", "password": "senha123", "api_key": "sua_chave_api" }

Resposta:

{ "token": "eyJhbGciOiJIUzI1NiIs...", "expires_in": 3600 }

📊 Puzzles

GET /puzzles

Lista todos os puzzles disponíveis.

[ { "id": 1, "title": "Áudio Misterioso", "difficulty": "easy", "solved": false } ]
GET /puzzles/{id}

Retorna detalhes de um puzzle específico.

{ "id": 1, "title": "Áudio Misterioso", "description": "Uma mensagem foi escondida...", "difficulty": "easy", "files": ["misterio.wav"], "hints": ["Analise o espectrograma entre 14kHz e 16kHz"] }
POST /puzzles/{id}/verify

Verifica se uma flag está correta.

{ "flag": "flag{exemplo}" }

Resposta:

{ "correct": true, "message": "Flag correta! Puzzle resolvido." }

👥 Agentes

GET /agents/ranking

Retorna o ranking dos top agentes.

[ { "position": 1, "username": "GrandMaster", "puzzles_solved": 50, "flags": 127, "avg_time": 300 } ]
GET /agents/{username}

Retorna perfil de um agente.

{ "username": "GrandMaster", "member_since": "2024-01-15", "puzzles_solved": 50, "achievements": 23, "last_flag": "2024-03-17T10:30:00Z" }

📈 Estatísticas

GET /stats/global

Estatísticas globais da plataforma.

{ "total_puzzles": 50, "total_solved": 1337, "total_agents": 1337, "total_flags": 3891, "by_level": { "beginner": 89, "intermediate": 76, "advanced": 62, "expert": 48, "master": 23 } }
GET /stats/realtime

Estatísticas em tempo real (WebSocket).

{ "agents_online": 42, "last_flag": "2 seconds ago", "current_record": "12s (Puzzle 02)" }

🛠️ Ferramentas

GET /tools/decode/base64

Decodifica Base64 via API.

GET /tools/decode/base64?string=Vm0wd2QyUXlVWGxV

Resposta:

{ "input": "Vm0wd2QyUXlVWGxV", "decoded": "use_steghide_na_imagem" }

⚠️ Códigos de Erro

400: Bad Request - Parâmetros inválidos 401: Unauthorized - Token ausente ou inválido 403: Forbidden - Sem permissão 404: Not Found - Recurso não encontrado 429: Too Many Requests - Limite de taxa excedido 500: Internal Server Error - Erro no servidor

📌 Rate Limits

- 100 requisições por minuto (autenticado) - 20 requisições por minuto (não autenticado) - Headers: X-RateLimit-Limit, X-RateLimit-Remaining

Para usar a API, obtenha sua chave em /community/forum
