# 👥 Times da Comunidade

## 🎯 O que são Times?
Times são grupos de agentes que se unem para resolver puzzles, participar de CTFs e compartilhar conhecimento.

## 🌟 Como Criar um Time

1. Ter pelo menos 3 membros interessados
2. Escolher um nome e tema
3. Definir líder e objetivos
4. Criar canal no Discord
5. Registrar no fórum
