🐘 PHP: Do Básico ao Avançado

40 minIniciante

📌 Introdução

PHP é a linguagem mais popular para desenvolvimento web backend. Cerca de 77% dos sites usam PHP.

⚠️ Erro Comum: Esquecer ponto e vírgula

❌ echo "Olá" // Erro!
echo "Olá" // Parse error
✅ Correção:
echo "Olá";

📝 Sintaxe Básica

<?php
// Variáveis
$nome = "João";
$idade = 25;
$preco = 19.90;

// Arrays
$frutas = ["maçã", "banana", "laranja"];

// Condicionais
if ($idade >= 18) {
    echo "Maior de idade";
}

// Loops
foreach ($frutas as $fruta) {
    echo $fruta;
}
?>
