⚙️ C/C++: Fundamentos

📖 45 min de leitura 🎯 Nível: Iniciante 💻 C/C++

📌 Introdução

C e C++ são linguagens poderosas para programação de sistemas, jogos e aplicações de alta performance. Neste tutorial, vamos aprender os fundamentos essenciais para começar.

📌 Hello World

#include <iostream>

int main() {
    std::cout << "Hello World!" << std::endl;
    return 0;
}

⚠️ Erro Comum: Esquecer std::

❌ Erro: 
cout << "teste";  // Erro! 'cout' não foi declarado

Problema: O compilador não reconhece 'cout' sem o namespace std.

✅ Correção: 
// Opção 1: Usar std::
std::cout << "teste";

// Opção 2: Adicionar using namespace std
using namespace std;
cout << "teste";

📝 Ponteiros

Ponteiros armazenam endereços de memória. São fundamentais em C/C++.

int x = 10;
int* ptr = &x;    // ptr armazena o endereço de x
*ptr = 20;        // x agora é 20

// Ponteiro para ponteiro
int** ptr2 = &ptr;
**ptr2 = 30;      // x agora é 30

📝 Exercício Prático

Crie uma função que troca os valores de duas variáveis usando ponteiros.

🎯 Solução: 
void trocar(int* a, int* b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 5, y = 10;
    trocar(&x, &y);
    // x = 10, y = 5
    return 0;
}
