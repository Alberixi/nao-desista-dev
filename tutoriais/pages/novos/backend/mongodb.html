🍃 MongoDB: NoSQL na Prática

30 minIntermediário

📌 Operações Básicas

// Inserir
db.usuarios.insertOne({
    nome: "João",
    email: "joao@email.com",
    idade: 25
})

// Buscar
db.usuarios.find({ idade: { $gt: 18 } })

// Atualizar
db.usuarios.updateOne(
    { nome: "João" },
    { $set: { idade: 26 } }
)

// Agregação
db.pedidos.aggregate([
    { $match: { status: "entregue" } },
    { $group: { _id: "$cliente", total: { $sum: "$valor" } } }
])

⚠️ Erro: Esquecer índices

❌ Buscas lentas em coleções grandes
db.usuarios.createIndex({ email: 1 }) // Índice único e rápido
