🗄️ SQL Injection

30 min Intermediário

📌 O que é?

Injeção de código SQL malicioso que pode expor ou deletar dados.

⚠️ Vulnerável

❌ Concatenação: 
$sql = "SELECT * FROM users WHERE user = '" . $_POST['user'] . "'";

Ataque: ' OR '1'='1' --

✅ Correção: 
$stmt = $pdo->prepare("SELECT * FROM users WHERE user = ?");
$stmt->execute([$_POST['user']]);
