🌓 Dark Mode com Variáveis CSS

📖 15 min de leitura 🎯 Nível: Iniciante 💻 Projeto prático

📌 Introdução

Dark mode se tornou um recurso essencial em aplicações modernas. Além de melhorar a experiência do usuário, pode reduzir o consumo de bateria em dispositivos OLED. Neste tutorial, vamos implementar um sistema de temas completo usando apenas CSS e JavaScript puro.

🎯 Objetivos

  • Entender variáveis CSS e como usá-las para temas
  • Implementar alternância de tema com JavaScript
  • Persistir a preferência do usuário com localStorage
  • Respeitar a preferência do sistema (prefers-color-scheme)
  • Criar transições suaves entre temas

⚠️ Erro Comum #1: Hardcoded de cores

❌ Erro: Definir cores fixas no CSS 
body {
    background-color: #ffffff;
    color: #000000;
}

.dark-mode {
    background-color: #000000;
    color: #ffffff;
}

Problema: Para cada elemento, você precisaria criar regras específicas para o dark mode, gerando código repetitivo e difícil de manter.

✅ Correção: Usar variáveis CSS 
:root {
    --bg-primary: #ffffff;
    --text-primary: #000000;
    --accent: #f97316;
}

body.dark {
    --bg-primary: #000000;
    --text-primary: #ffffff;
    --accent: #38bdf8;
}

body {
    background-color: var(--bg-primary);
    color: var(--text-primary);
}

.button {
    background-color: var(--accent);
}

⚠️ Erro Comum #2: Não persistir a preferência

❌ Erro: Tema volta ao padrão ao recarregar 
// O tema é perdido quando a página é recarregada
function toggleTheme() {
    document.body.classList.toggle('dark');
}

Problema: O usuário perde a preferência ao navegar entre páginas ou recarregar.

✅ Correção: Salvar no localStorage 
function toggleTheme() {
    const isDark = document.body.classList.toggle('dark');
    localStorage.setItem('theme', isDark ? 'dark' : 'light');
}

function loadTheme() {
    const savedTheme = localStorage.getItem('theme');
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    
    if (savedTheme === 'dark' || (!savedTheme && prefersDark)) {
        document.body.classList.add('dark');
    }
}

loadTheme();

⚠️ Erro Comum #3: Transições abruptas

❌ Erro: Mudança instantânea de cores 
// Sem transição, a mudança é abrupta
body {
    background-color: var(--bg-primary);
    color: var(--text-primary);
}

Problema: A mudança de tema acontece instantaneamente, sem fluidez.

✅ Correção: Adicionar transições 
body {
    background-color: var(--bg-primary);
    color: var(--text-primary);
    transition: background-color 0.3s ease, color 0.2s ease;
}

/* Otimização para performance */
* {
    transition: background-color 0.3s ease, border-color 0.2s ease;
}

🚀 Evolução: Do básico ao avançado

1. Versão básica (apenas CSS)

@media (prefers-color-scheme: dark) {
    body {
        background-color: #000;
        color: #fff;
    }
}

2. Com toggle manual (JavaScript)

const toggle = document.getElementById('theme-toggle');
toggle.addEventListener('click', () => {
    document.body.classList.toggle('dark');
});

3. Versão completa com persistência e preferência do sistema

// Detectar preferência do sistema
const prefersDark = window.matchMedia('(prefers-color-scheme: dark)');

// Função para aplicar tema
function applyTheme(theme) {
    if (theme === 'dark') {
        document.body.classList.add('dark');
    } else {
        document.body.classList.remove('dark');
    }
    localStorage.setItem('theme', theme);
}

// Carregar tema salvo ou detectar sistema
const savedTheme = localStorage.getItem('theme');
if (savedTheme) {
    applyTheme(savedTheme);
} else {
    applyTheme(prefersDark.matches ? 'dark' : 'light');
}

// Ouvir mudanças no sistema
prefersDark.addEventListener('change', (e) => {
    if (!localStorage.getItem('theme')) {
        applyTheme(e.matches ? 'dark' : 'light');
    }
});
🎯 Evolução:

Comece com um simples media query, evolua para um toggle manual e finalize com um sistema completo que respeita a preferência do usuário e do sistema!

💡 Dicas Avançadas

  • CSS-in-JS: Em frameworks como React, use Context API para gerenciar temas globalmente
  • Imagens adaptativas: Use <picture> com media queries para carregar imagens específicas para dark mode
  • Performance: Evite transições em propriedades que causam repaint (evite box-shadow)
  • Subtemas: Crie múltiplas variações de cores (azul, verde, roxo) além do claro/escuro

Exemplo com múltiplos temas

body[data-theme="light"] {
    --bg: #fff;
    --text: #000;
}

body[data-theme="dark"] {
    --bg: #000;
    --text: #fff;
}

body[data-theme="blue"] {
    --bg: #1e3a8a;
    --text: #e0f2fe;
}

function setTheme(theme) {
    document.body.setAttribute('data-theme', theme);
    localStorage.setItem('theme', theme);
}
