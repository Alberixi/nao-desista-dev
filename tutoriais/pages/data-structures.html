🔗 Estrutura de Dados: Linked List (Lista Encadeada)

📖 25 min de leitura 🎯 Nível: Intermediário 💻 JavaScript • Algoritmos 📊 Complexidade O(n)

📌 Introdução

Listas encadeadas são uma das estruturas de dados fundamentais na ciência da computação. Diferente dos arrays, que armazenam elementos em posições contíguas de memória, as listas encadeadas armazenam elementos em nós independentes, cada um apontando para o próximo. Isso oferece flexibilidade para inserções e remoções, mas com trade-offs em acesso aleatório.

Neste tutorial, vamos implementar do zero uma lista encadeada em JavaScript, entender seus pontos fortes e fracos, e aprender com erros comuns.

🎯 Objetivos

  • Entender o conceito de nós e referências
  • Implementar operações básicas: inserção, remoção, busca
  • Comparar listas encadeadas com arrays
  • Resolver problemas clássicos com listas encadeadas
  • Analisar complexidade de tempo e espaço

📊 Visualização da Lista Encadeada

[Head] → [10] → [20] → [30] → [40] → null
          ↑       ↑       ↑       ↑
        Node    Node    Node    Node
      (value) (value) (value) (value)

Cada nó contém um valor e uma referência (ponteiro) para o próximo nó. O último nó aponta para null, indicando o fim da lista.

⚠️ Erro Comum #1: Perder a referência da cabeça da lista

❌ Erro: Manipular a lista sem manter referência ao head 
class LinkedList {
    constructor() {
        this.head = null;
    }
    
    // ERRO: Perde a referência original
    addToFront(value) {
        let current = this.head;  // Guarda referência
        let newNode = new Node(value);
        newNode.next = current;
        current = newNode;  // Isso não altera this.head!
    }
}

Problema: A lista parece funcionar dentro do método, mas a propriedade head não é atualizada.

✅ Correção: Atualizar this.head diretamente 
class LinkedList {
    constructor() {
        this.head = null;
    }
    
    addToFront(value) {
        const newNode = new Node(value);
        newNode.next = this.head;
        this.head = newNode;  // Atualiza a cabeça da lista
    }
    
    // Para visualizar
    print() {
        let current = this.head;
        const values = [];
        while (current) {
            values.push(current.value);
            current = current.next;
        }
        console.log(values.join(' → '));
    }
}

⚠️ Erro Comum #2: Esquecer de lidar com casos extremos

❌ Erro: Assumir que a lista sempre tem elementos 
removeFromEnd() {
    let current = this.head;
    // Se a lista estiver vazia, this.head é null
    while (current.next.next) {  // TypeError: cannot read property 'next' of null
        current = current.next;
    }
    current.next = null;
}

Problema: O código quebra quando a lista está vazia ou tem apenas um elemento.

✅ Correção: Validar casos extremos 
removeFromEnd() {
    // Caso 1: Lista vazia
    if (!this.head) {
        return null;
    }
    
    // Caso 2: Apenas um elemento
    if (!this.head.next) {
        const removed = this.head.value;
        this.head = null;
        return removed;
    }
    
    // Caso 3: Múltiplos elementos
    let current = this.head;
    while (current.next.next) {
        current = current.next;
    }
    const removed = current.next.value;
    current.next = null;
    return removed;
}

⚠️ Erro Comum #3: Complexidade desnecessária em buscas

❌ Erro: Percorrer a lista múltiplas vezes 
// Buscar um elemento e depois removê-lo
find(value) {
    let current = this.head;
    let index = 0;
    while (current) {
        if (current.value === value) {
            return index;
        }
        current = current.next;
        index++;
    }
    return -1;
}

removeByValue(value) {
    const index = this.find(value);  // Primeira busca O(n)
    if (index === 0) {
        this.head = this.head.next;
    } else if (index > 0) {
        let current = this.head;
        for (let i = 0; i < index - 1; i++) {  // Segunda busca O(n)
            current = current.next;
        }
        current.next = current.next.next;
    }
}

Problema: Duas passadas pela lista, complexidade O(2n) = O(n), mas ineficiente na prática.

✅ Correção: Remover em uma única passada 
removeByValue(value) {
    if (!this.head) return false;
    
    // Remover da cabeça
    if (this.head.value === value) {
        this.head = this.head.next;
        return true;
    }
    
    // Remover do meio/fim
    let current = this.head;
    while (current.next) {
        if (current.next.value === value) {
            current.next = current.next.next;
            return true;
        }
        current = current.next;
    }
    
    return false; // Elemento não encontrado
}

🚀 Evolução: Implementação completa de Linked List

1. Estrutura do Nó e Lista Básica

class Node {
    constructor(value) {
        this.value = value;
        this.next = null;
    }
}

class LinkedList {
    constructor() {
        this.head = null;
        this.size = 0;
    }
    
    // Adicionar ao final
    append(value) {
        const newNode = new Node(value);
        
        if (!this.head) {
            this.head = newNode;
        } else {
            let current = this.head;
            while (current.next) {
                current = current.next;
            }
            current.next = newNode;
        }
        this.size++;
        return this;
    }
    
    // Adicionar ao início
    prepend(value) {
        const newNode = new Node(value);
        newNode.next = this.head;
        this.head = newNode;
        this.size++;
        return this;
    }
    
    // Inserir em posição específica
    insertAt(value, index) {
        if (index < 0 || index > this.size) {
            throw new Error('Índice inválido');
        }
        
        if (index === 0) {
            return this.prepend(value);
        }
        
        const newNode = new Node(value);
        let current = this.head;
        for (let i = 0; i < index - 1; i++) {
            current = current.next;
        }
        
        newNode.next = current.next;
        current.next = newNode;
        this.size++;
        return this;
    }
    
    // Remover por índice
    removeAt(index) {
        if (index < 0 || index >= this.size || !this.head) {
            return null;
        }
        
        let removed;
        
        if (index === 0) {
            removed = this.head.value;
            this.head = this.head.next;
        } else {
            let current = this.head;
            for (let i = 0; i < index - 1; i++) {
                current = current.next;
            }
            removed = current.next.value;
            current.next = current.next.next;
        }
        
        this.size--;
        return removed;
    }
    
    // Buscar valor
    find(value) {
        let current = this.head;
        let index = 0;
        
        while (current) {
            if (current.value === value) {
                return index;
            }
            current = current.next;
            index++;
        }
        
        return -1;
    }
    
    // Acessar por índice
    get(index) {
        if (index < 0 || index >= this.size) {
            return null;
        }
        
        let current = this.head;
        for (let i = 0; i < index; i++) {
            current = current.next;
        }
        
        return current.value;
    }
    
    // Converter para array
    toArray() {
        const result = [];
        let current = this.head;
        while (current) {
            result.push(current.value);
            current = current.next;
        }
        return result;
    }
    
    // Inverter lista
    reverse() {
        let prev = null;
        let current = this.head;
        let next = null;
        
        while (current) {
            next = current.next;
            current.next = prev;
            prev = current;
            current = next;
        }
        
        this.head = prev;
        return this;
    }
    
    // Verificar se está vazia
    isEmpty() {
        return this.size === 0;
    }
    
    // Tamanho da lista
    getSize() {
        return this.size;
    }
    
    // Limpar lista
    clear() {
        this.head = null;
        this.size = 0;
        return this;
    }
    
    // Print formatado
    print() {
        console.log(this.toArray().join(' → '));
        return this;
    }
}

// Testes
const list = new LinkedList();
list.append(10).append(20).append(30);
list.prepend(5);
list.insertAt(15, 2);
list.print(); // 5 → 10 → 15 → 20 → 30
console.log(list.find(20)); // 3
console.log(list.get(2)); // 15
list.removeAt(1);
list.print(); // 5 → 15 → 20 → 30
list.reverse();
list.print(); // 30 → 20 → 15 → 5

2. Lista Duplamente Encadeada (Avançado)

class DoublyNode {
    constructor(value) {
        this.value = value;
        this.prev = null;
        this.next = null;
    }
}

class DoublyLinkedList {
    constructor() {
        this.head = null;
        this.tail = null;
        this.size = 0;
    }
    
    append(value) {
        const newNode = new DoublyNode(value);
        
        if (!this.head) {
            this.head = newNode;
            this.tail = newNode;
        } else {
            newNode.prev = this.tail;
            this.tail.next = newNode;
            this.tail = newNode;
        }
        
        this.size++;
        return this;
    }
    
    prepend(value) {
        const newNode = new DoublyNode(value);
        
        if (!this.head) {
            this.head = newNode;
            this.tail = newNode;
        } else {
            newNode.next = this.head;
            this.head.prev = newNode;
            this.head = newNode;
        }
        
        this.size++;
        return this;
    }
    
    removeFromEnd() {
        if (!this.tail) return null;
        
        const removed = this.tail.value;
        
        if (this.tail === this.head) {
            this.head = null;
            this.tail = null;
        } else {
            this.tail = this.tail.prev;
            this.tail.next = null;
        }
        
        this.size--;
        return removed;
    }
    
    printForward() {
        const values = [];
        let current = this.head;
        while (current) {
            values.push(current.value);
            current = current.next;
        }
        console.log(values.join(' ↔ '));
    }
    
    printBackward() {
        const values = [];
        let current = this.tail;
        while (current) {
            values.push(current.value);
            current = current.prev;
        }
        console.log(values.join(' ↔ '));
    }
}

// Teste
const doublyList = new DoublyLinkedList();
doublyList.append(1).append(2).append(3);
doublyList.prepend(0);
doublyList.printForward();  // 0 ↔ 1 ↔ 2 ↔ 3
doublyList.printBackward(); // 3 ↔ 2 ↔ 1 ↔ 0

Comece com listas simplesmente encadeadas, depois evolua para duplamente encadeadas para navegação bidirecional. Domine as operações básicas antes de implementar estruturas mais complexas.

💡 Comparação: Linked List vs Array

┌───────────────────┬──────────────────┬────────────────────┐
│    Operação       │    Array         │  Linked List       │
├───────────────────┼──────────────────┼────────────────────┤
│ Acesso por índice │ O(1) ✓           │ O(n) ✗            │
│ Inserção no fim   │ O(1) ✓           │ O(1) ✓            │
│ Inserção no início│ O(n) ✗           │ O(1) ✓            │
│ Remoção no início │ O(n) ✗           │ O(1) ✓            │
│ Inserção no meio  │ O(n)             │ O(n) (busca + O(1))│
│ Busca por valor   │ O(n)             │ O(n)               │
│ Uso de memória    │ Menor (contíguo) │ Maior (ponteiros)  │
└───────────────────┴──────────────────┴────────────────────┘

Quando usar Linked List: Inserções/remoções frequentes no início/meio, quando o tamanho é dinâmico e imprevisível.

Quando usar Array: Acesso aleatório frequente, quando o tamanho é conhecido e estável.

💡 Dicas Avançadas

  • Detecção de ciclo: Use algoritmo de Floyd (tartaruga e lebre) para detectar loops
  • Merge de listas ordenadas: Intercale duas listas mantendo a ordenação
  • Encontrar elemento do meio: Use dois ponteiros (um anda 1 passo, outro 2 passos)
  • Skip List: Estrutura mais avançada que combina linked lists com busca binária

Exemplo: Detecção de ciclo (Floyd's Cycle Detection)

hasCycle() {
    if (!this.head) return false;
    
    let slow = this.head;
    let fast = this.head;
    
    while (fast && fast.next) {
        slow = slow.next;      // 1 passo
        fast = fast.next.next; // 2 passos
        
        if (slow === fast) {
            return true; // Ciclo detectado
        }
    }
    
    return false;
}
