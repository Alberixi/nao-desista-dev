🎨 CSS Grid: Layouts Modernos

📖 15 min de leitura 🎯 Nível: Intermediário 💻 Código prático

📌 Introdução

CSS Grid é um dos sistemas de layout mais poderosos do CSS moderno. Ele permite criar layouts bidimensionais complexos com poucas linhas de código. Neste tutorial, vamos aprender desde o básico até técnicas avançadas, incluindo erros comuns e como corrigi-los.

🎯 Objetivos

  • Entender os conceitos fundamentais do CSS Grid
  • Criar layouts responsivos com Grid
  • Identificar e corrigir erros comuns
  • Aplicar técnicas de evolução e otimização

⚠️ Erro Comum #1: Não definir o container como grid

❌ Erro: 
.container {
    /* Esqueceu de definir display: grid */
    grid-template-columns: 1fr 1fr;
}

Problema: As propriedades do Grid não funcionam porque o container não foi definido como grid.

✅ Correção: 
.container {
    display: grid; /* ESSENCIAL! */
    grid-template-columns: 1fr 1fr;
    gap: 1rem;
}

⚠️ Erro Comum #2: Unidades fixas sem responsividade

❌ Erro: 
.grid {
    display: grid;
    grid-template-columns: 300px 300px 300px; /* Fixo, quebra em telas menores */
}

Problema: Layout quebra em dispositivos móveis e não se adapta.

✅ Correção: 
.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 1rem;
}

Explicação: auto-fit e minmax() criam colunas flexíveis que se ajustam automaticamente.

⚠️ Erro Comum #3: Não usar gap corretamente

❌ Erro: 
.grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
    margin: -0.5rem; /* Hack antigo para espaçamento */
}
.grid-item {
    margin: 0.5rem;
}

Problema: Usar margin para espaçamento entre itens do grid causa problemas de alinhamento e overflow.

✅ Correção: 
.grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 1rem; /* Propriedade moderna e correta! */
}

🚀 Evolução: Do básico ao avançado

1. Layout básico

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    gap: 1rem;
}

2. Layout responsivo com media queries

.container {
    display: grid;
    grid-template-columns: 1fr;
    gap: 1rem;
}

@media (min-width: 768px) {
    .container {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (min-width: 1024px) {
    .container {
        grid-template-columns: repeat(3, 1fr);
    }
}

3. Layout avançado com Grid Areas

.container {
    display: grid;
    grid-template-areas: 
        "header header header"
        "sidebar main main"
        "footer footer footer";
    grid-template-columns: 250px 1fr 1fr;
    gap: 1rem;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }
🎯 Evolução:

Comece com layouts simples e vá adicionando complexidade gradualmente. Sempre priorize a responsividade desde o início!

💡 Dicas Avançadas

  • Grid implícito vs explícito: Use grid-auto-rows para controlar linhas adicionais
  • Alinhamento: place-items centraliza itens facilmente
  • Performance: Grid é mais performático que Flexbox para layouts 2D
  • Subgrid: (CSS Grid Level 2) Permite que itens herdem a grade do pai

📝 Exercício Prático

Crie um portfólio responsivo com as seguintes características:

  • Header com navegação
  • Grid de projetos (3 colunas no desktop, 2 no tablet, 1 no mobile)
  • Footer com informações
  • Utilize grid-template-areas para organizar
🎯 Desafio extra: Adicione animações suaves nos cards ao hover usando transitions.
