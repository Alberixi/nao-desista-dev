# 🌐 Workshop: Wireshark Master

## Sobre o Workshop
Workshop avançado de 6 horas sobre análise de tráfego com Wireshark.

## 📅 Agenda

### Parte 1: Fundamentos (1h)
- Captura de pacotes
- Filtros básicos
- Estatísticas

### Parte 2: Filtros Avançados (1.5h)
- Filtros de exibição complexos
- Filtros de captura
- Expressões regulares

### Parte 3: Análise de Protocolos (1.5h)
- HTTP/HTTPS
- TCP/UDP
- DNS
- SMB

### Parte 4: Segurança (1h)
- Detecção de ataques
- Análise de malware
- Exfiltração de dados

### Parte 5: Prática (1h)
- PCAPs reais
- CTF de rede

## 🛠️ Ferramentas Utilizadas
- Wireshark
- tshark
- tcpdump
- NetworkMiner
