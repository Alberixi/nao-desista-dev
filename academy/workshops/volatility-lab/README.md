# 🧠 Workshop: Volatility Lab

## Sobre o Workshop
Workshop prático de 5 horas sobre análise de memória com Volatility.

## 📅 Agenda

### Parte 1: Fundamentos (1h)
- O que é Volatility
- Perfis de memória
- Plugins básicos

### Parte 2: Análise de Processos (1h)
- pslist, psscan, pstree
- Detecção de processos ocultos
- Dump de processos

### Parte 3: Análise de Rede (1h)
- netscan, connections
- sockets
- Detecção de C2

### Parte 4: Análise de Malware (1.5h)
- malfind
- ldrmodules
- detecção de injeção

### Parte 5: Extração de Dados (0.5h)
- hashdump
- lsadump
- dumpfiles

## 🛠️ Ferramentas Utilizadas
- Volatility 2 e 3
- Rekall
- DumpIt

- [Slides](slides.pdf)

## 🎯 Objetivos
- Identificar processos maliciosos
- Extrair indicadores de comprometimento
- Recuperar hashes e senhas
- Analisar injeção de código
