# 🐧 Workshop: Linux Forensics

## Sobre o Workshop
Workshop prático de 4 horas sobre análise forense em sistemas Linux.

## 📅 Agenda

### Parte 1: Fundamentos (1h)
- Estrutura de diretórios Linux
- Sistemas de arquivos (ext4, xfs)
- Logs do sistema (/var/log)

### Parte 2: Coleta de Evidências (1h)
- Aquisição de memória (LiME)
- Coleta de discos (dd, dcfldd)
- Coleta de logs (journalctl)

### Parte 3: Análise (1.5h)
- Análise de logs (grep, awk, sed)
- Recuperação de arquivos (foremost, scalpel)
- Timeline analysis (fls, mactime)

### Parte 4: Caso Prático (0.5h)
- Investigação de intrusão real
- Criação de relatório

## 🛠️ Ferramentas Utilizadas
- The Sleuth Kit
- Volatility
- Foremost
- Autopsy
- Wireshark

## 📝 Material
- [Slides](slides.pdf)
- [VM para prática](vm.ova)
- [Casos de exemplo](cases/)

## 🎯 Objetivos
- Coletar evidências forenses
- Analisar logs do sistema
- Recuperar arquivos deletados
- Criar linha do tempo da intrusão
- Gerar relatório técnico
