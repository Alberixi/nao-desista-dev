# ⚙️ Engenharia Reversa

## Sobre o Curso
Curso aprofundado de engenharia reversa de software, ensinando a desmontar e entender binários.

## 📚 Conteúdo Programático

### Módulo 1: Fundamentos
- Linguagem Assembly (x86/x64)
- Pilha e registradores
- Endianness

### Módulo 2: Ferramentas
- Disassemblers: Ghidra, IDA Pro
- Debuggers: GDB, x64dbg, OllyDbg
- Hex editors

### Módulo 3: Análise de Binários
- PE/ELF format
- Sections e headers
- Import/Export tables

### Módulo 4: Técnicas de RE
- Análise estática
- Análise dinâmica
- Patching de binários

### Módulo 5: Desafios
- Crackmes
- CTF challenges
- Malware unpacking

## 🎯 Projetos Práticos
1. Analisar crackme simples
2. Encontrar senha hardcoded em binário
3. Patch de condicional para bypass de login
