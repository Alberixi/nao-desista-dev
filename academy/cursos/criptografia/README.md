# 🔐 Criptografia

## Sobre o Curso
Curso abrangente sobre criptografia, desde conceitos básicos até implementações práticas.

## 📚 Conteúdo Programático

### Módulo 1: Fundamentos
- História da criptografia
- Terminologia básica
- Cifras clássicas (César, Vigenère, Enigma)

### Módulo 2: Hash Functions
- MD5, SHA-1, SHA-256, SHA-3
- Aplicações de hash
- Colisões e segurança

### Módulo 3: Criptografia Simétrica
- AES, DES, ChaCha20
- Modos de operação (ECB, CBC, GCM)
- Implementações práticas

### Módulo 4: Criptografia Assimétrica
- RSA, ECC, Diffie-Hellman
- Trocas de chaves
- Assinaturas digitais

### Módulo 5: Aplicações
- TLS/SSL
- PGP/GPG
- Blockchain e criptomoedas

## 🎯 Projetos Práticos
1. Implementar cifra de César em Python
2. Quebrar hash MD5 com dicionário
3. Simular troca de chaves Diffie-Hellman
