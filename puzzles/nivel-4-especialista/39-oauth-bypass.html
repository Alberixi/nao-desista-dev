> 39: OAUTH BYPASS

ESPECIALISTA

Uma implementação OAuth vulnerável permite login sem validação adequada do state. Bypasse a tela de login.

FLUXO OAUTH

GET /authorize?client_id=app&redirect_uri=https://cliente.com/callback&response_type=code&state=123

Dica: O parâmetro state não está sendo validado. Você pode usar um CSRF para logar como admin.

DICA: Crie uma URL de autorização, troque o code e ignore o state.
