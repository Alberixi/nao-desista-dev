> 36: XSS CHALLENGE

ESPECIALISTA

Um site vulnerável a XSS refletido. Execute um alerta com o document.cookie para capturar a flag.

SITE VULNERÁVEL (SIMULAÇÃO)

TESTAR XSS (EXEMPLO)

Parâmetro vulnerável: ?search=<script>alert(document.cookie)</script>

// Código PHP vulnerável (ilustrativo): <?php $search = $_GET['search'] ?? ''; echo "Você buscou por: " . $search; // SEM SANITIZAÇÃO! ?>

O que acontece:

  • 1️⃣ O parâmetro search não é sanitizado
  • 2️⃣ O valor é inserido diretamente no HTML com echo
  • 3️⃣ Se o valor contiver <script>, ele será executado!
DICA: O cookie contém a flag. Use <script>alert(document.cookie)</script>

Em um sistema real, você veria os cookies do usuário!
