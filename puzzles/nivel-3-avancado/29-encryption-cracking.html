> 29: ENCRYPTION CRACKING

AVANÇADO

Um algoritmo de criptografia personalizado foi usado. O texto cifrado e o código fonte estão disponíveis. Descubra a flag.

TEXTO CIFRADO (hex)

4e 61 6f 20 64 65 73 69 73 74 61 20 64 65 76 21

CÓDIGO FONTE

def encrypt(text, key): result = [] for i, c in enumerate(text): result.append(chr(ord(c) ^ key[i % len(key)])) return ''.join(result) # key = [0x66, 0x6c, 0x61, 0x67]

Dica: A criptografia é XOR com chave repetitiva. A chave está no código.

DICA: A chave é "flag". Aplique XOR novamente para decifrar.
