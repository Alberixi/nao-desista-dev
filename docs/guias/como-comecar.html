> 🚀 COMO COMEÇAR - GUIA DO INVESTIGADOR

Bem-vindo, agente! Este guia vai te ajudar a navegar pela plataforma Não Desista Dev e começar sua jornada de investigação digital.

💡 DICA: A persistência é a chave. Não desista no primeiro obstáculo!

📋 Pré-requisitos

Antes de começar, tenha em mãos:

  • Um navegador moderno (Chrome, Firefox, Edge)
  • Curiosidade e persistência
  • Conhecimentos básicos de terminal (útil mas não obrigatório)

🎯 Navegando pela Plataforma

Página Inicial (index.html)

A página principal contém:

  • Terminal Header: Status do sistema e contador de agentes
  • Hero Section: Boas-vindas e terminal informativo
  • Stats Grid: Estatísticas gerais (puzzles, ferramentas, flags)
  • Puzzles Grid: Todos os 50 desafios organizados por nível
  • Ferramentas: Links para ferramentas úteis
  • Documentação: Guias e referências
  • Terminal Interativo: Simulador de terminal Linux

🛠️ Como Usar os Puzzles

  1. Acesse um puzzle clicando no card correspondente
  2. Leia a descrição do desafio
  3. Baixe os arquivos disponíveis (quando houver)
  4. Use as ferramentas recomendadas
  5. Descubra a flag no formato flag{...}
  6. Submeta a flag no campo de verificação

💡 Dicas Importantes

  • 🔍 Inspecione o código fonte (F12) - sempre há pistas
  • 📚 Consulte a documentação antes de desistir
  • 🛠️ Use as ferramentas disponíveis na pasta /tools
  • 🤝 Participe da comunidade para trocar ideias
  • 🏆 Colecione flags e veja seu progresso

🐧 Terminal Interativo

O terminal na página inicial simula comandos Linux úteis:

help Lista todos os comandos
ls Lista arquivos
cat Lê arquivos
ping Testa conectividade
status Verifica progresso
hint Dica do dia

📈 Progresso e Conquistas

Seu progresso é salvo localmente no navegador. Ao resolver um puzzle:

  1. A flag é verificada
  2. O puzzle é marcado como "RESOLVIDO"
  3. O contador de flags aumenta
  4. O status do sistema é atualizado
⚠️ Atenção: Se você limpar os dados do navegador, seu progresso será perdido!

🆘 Precisa de Ajuda?

  • 📖 Leia os tutoriais específicos
  • 💬 Acesse o fórum da comunidade
  • 🎯 Use as dicas disponíveis em cada puzzle
  • 🧪 Pratique nos laboratórios

🎓 Próximos Passos

  1. Comece pelos puzzles 01-10 (Iniciante)
  2. Avance gradualmente pelos níveis
  3. Explore as ferramentas
  4. Participe dos CTFs semanais
  5. Compartilhe suas conquistas
🏆 Lembre-se: A jornada é longa, mas cada flag encontrada é uma vitória!
