> COMANDOS LINUX ESSENCIAIS PARA FORENSE

💡 Dica: Estes comandos são fundamentais para análise forense e investigação digital.

📁 Navegação e Arquivos

ls -la Lista todos os arquivos (incluindo ocultos) com detalhes
cd /pasta Navega para o diretório especificado
pwd Mostra o caminho do diretório atual
find . -name "*.txt" Procura arquivos por nome
tree Exibe estrutura de diretórios em árvore

🔍 Visualização e Análise

cat arquivo.txt Mostra conteúdo do arquivo
less arquivo.txt Visualiza paginado (setas para navegar, q para sair)
head -n 20 arquivo.txt Mostra as primeiras 20 linhas
tail -n 20 arquivo.txt Mostra as últimas 20 linhas
strings arquivo.bin Extrai texto legível de binários
hexdump -C arquivo.bin Visualização hexadecimal
xxd arquivo.bin Outra ferramenta de hexdump

🔐 Permissões e Propriedades

chmod +x script.sh Adiciona permissão de execução
chmod 644 arquivo.txt Define permissões específicas
chown usuario:grupo arquivo Altera proprietário
stat arquivo.txt Mostra metadados completos
file arquivo Identifica o tipo do arquivo

🔎 Busca e Filtros

grep "palavra" arquivo.txt Busca texto em arquivos
grep -r "flag" ./ Busca recursiva em diretórios
grep -i "case" Ignora maiúsculas/minúsculas
awk '{print $1}' Processa colunas de texto
sed 's/antigo/novo/g' Substitui texto
sort arquivo.txt Ordena linhas
uniq -c Conta linhas únicas
wc -l arquivo.txt Conta linhas

📦 Discos e Sistema de Arquivos

df -h Espaço em disco (formato humano)
du -sh * Tamanho de pastas/arquivos
mount Mostra partições montadas
fdisk -l Lista partições
dd if=/dev/sda of=imagem.dd Cria imagem de disco
foremost -i imagem.dd -o output/ Recupera arquivos deletados
scalpel imagem.dd -o output/ Carving de arquivos

🌐 Rede

ifconfig / ip a Mostra interfaces de rede
netstat -tulpn Conexões e portas abertas
ss -tulpn Alternativa moderna ao netstat
nmap -p 1-1000 192.168.1.1 Scanner de portas
tcpdump -i eth0 -w captura.pcap Captura pacotes
wireshark captura.pcap Analisa pacotes (GUI)
ping google.com Testa conectividade
traceroute google.com Rota até o destino

🔄 Processos e Sistema

ps aux Lista todos os processos
top / htop Monitor de processos em tempo real
kill -9 PID Mata processo forçadamente
uname -a Informações do sistema
dmesg Mensagens do kernel
journalctl -xe Logs do sistema (systemd)

🎯 Comandos Específicos para Forense

md5sum arquivo Calcula hash MD5
sha256sum arquivo Calcula hash SHA256
exiftool imagem.jpg Extrai metadados EXIF
binwalk imagem.png Analisa arquivos embutidos
steghide extract -sf imagem.jpg Extrai dados escondidos
zsteg imagem.png Detecta LSB em PNG
volatility -f memory.raw imageinfo Analisa dump de memória
📚 Exercício: Tente usar o comando strings em um arquivo binário para encontrar texto oculto.
