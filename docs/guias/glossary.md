# 📚 Glossário do Investigador

## 🔐 Criptografia

| Termo | Definição |
|-------|-----------|
| **AES** | Advanced Encryption Standard - Algoritmo de criptografia simétrica |
| **Base64** | Codificação que representa dados binários em texto ASCII |
| **Cifra de César** | Substituição onde cada letra é deslocada um número fixo de posições |
| **Hash** | Função unidirecional que gera uma impressão digital dos dados |
| **MD5** | Algoritmo de hash de 128 bits (considerado inseguro) |
| **RSA** | Criptografia assimétrica baseada em fatoração de números primos |
| **ROT13** | Cifra de César com deslocamento de 13 posições |
| **SHA** | Secure Hash Algorithm - Família de funções hash |
| **XOR** | Operação lógica "ou exclusivo" usada em criptografia |

## 🕵️ Esteganografia

| Termo | Definição |
|-------|-----------|
| **Esteganografia** | Técnica de esconder informações dentro de outros arquivos |
| **LSB** | Least Significant Bit - Bit menos significativo, usado para esconder dados em imagens |
| **Metadados** | Dados sobre dados (EXIF em imagens, propriedades de arquivos) |
| **Espectrograma** | Representação visual do espectro de frequências de um áudio |
| **Carving** | Técnica de recuperar arquivos baseada em suas assinaturas |

## 🔍 Forense Digital

| Termo | Definição |
|-------|-----------|
| **Dump de Memória** | Cópia do conteúdo da RAM para análise |
| **Imageamento** | Cópia bit-a-bit de um disco ou partição |
| **IoC** | Indicator of Compromise - Evidência de invasão |
| **PCAP** | Formato de arquivo para captura de pacotes de rede |
| **Volatility** | Framework para análise de memória RAM |
| **Wireshark** | Ferramenta de análise de protocolos de rede |

## 🌐 Rede

| Termo | Definição |
|-------|-----------|
| **DNS** | Domain Name System - Traduz nomes para IPs |
| **HTTP** | Protocolo de transferência de hipertexto |
| **HTTPS** | Versão segura do HTTP |
| **IP** | Internet Protocol - Endereço de um dispositivo na rede |
| **Porta** | Ponto de comunicação em um dispositivo |
| **TCP** | Protocolo de transmissão confiável |
| **UDP** | Protocolo de transmissão não confiável |

## 🛡️ Segurança Web

| Termo | Definição |
|-------|-----------|
| **CSRF** | Cross-Site Request Forgery - Forja de requisições |
| **IDOR** | Insecure Direct Object Reference - Acesso direto a objetos |
| **JWT** | JSON Web Token - Token de autenticação |
| **OAuth** | Protocolo de autorização |
| **SQLi** | SQL Injection - Injeção de comandos SQL |
| **XSS** | Cross-Site Scripting - Injeção de scripts |

## 💻 Comandos Linux

| Comando | Função |
|---------|--------|
| **cat** | Mostra conteúdo de arquivo |
| **chmod** | Altera permissões |
| **grep** | Busca texto em arquivos |
| **ls** | Lista arquivos e pastas |
| **strings** | Extrai texto de binários |
| **xxd** | Visualização hexadecimal |

## 📁 Tipos de Arquivo

| Extensão | Descrição |
|----------|-----------|
| **.pcap** | Captura de pacotes de rede |
| **.raw/.mem** | Dump de memória RAM |
| **.dd/.img** | Imagem de disco |
| **.elf** | Executável Linux |
| **.exe** | Executável Windows |

## 🎯 Flags

| Termo | Definição |
|-------|-----------|
| **Flag** | String no formato `flag{...}` que comprova a solução do puzzle |
| **Flag Local** | Flag encontrada no próprio sistema (usuário) |
| **Flag Root** | Flag de privilégios máximos (administrador) |

## 🤝 Comunidade

| Termo | Definição |
|-------|-----------|
| **CTF** | Capture The Flag - Competição de segurança |
| **OSINT** | Open Source Intelligence - Inteligência de fontes abertas |
| **APT** | Advanced Persistent Threat - Ameaça persistente avançada |
| **Zero Day** | Vulnerabilidade desconhecida pelo fabricante |
