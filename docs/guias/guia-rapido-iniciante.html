🐧 Comandos Linux que Você Precisa Saber

ls -la Lista arquivos (incluindo ocultos)
cat arquivo Mostra conteúdo do arquivo
grep "texto" * Busca texto em arquivos
strings arquivo Extrai texto de binários
file arquivo Identifica tipo do arquivo
hexdump -C arquivo Visualização hexadecimal
find . -name "*.txt" Procura arquivos por nome
exiftool imagem.jpg Metadados da imagem

📚 Codificações Comuns

Base64

Começa com letras maiúsculas/minúsculas e números, termina com =

echo "texto" | base64

Hexadecimal

Apenas 0-9 e A-F, geralmente em pares

xxd -r

ROT13

Deslocamento de 13 letras (a→n, b→o)

tr 'A-Za-z' 'N-ZA-Mn-za-m'

Binário

Apenas 0 e 1, grupos de 8 bits

Converter para ASCII

🔍 Técnicas Rápidas por Tipo de Puzzle

🖼️ Imagens

  1. exiftool imagem.jpg - Metadados
  2. steghide extract -sf imagem.jpg - Dados escondidos
  3. binwalk imagem.png - Arquivos embutidos
  4. zsteg imagem.png - LSB em PNG

🎵 Áudio

  1. Audacity → View → Spectrogram - Imagem nas frequências
  2. strings audio.wav | grep flag - Texto oculto

📄 Arquivos

  1. strings arquivo | grep flag - Texto legível
  2. binwalk arquivo - Arquivos embutidos
  3. foremost -i arquivo -o output/ - Carving

🌐 Rede (PCAP)

  1. tshark -r captura.pcap -Y "http" - Filtrar HTTP
  2. tcpflow -r captura.pcap - Reconstruir fluxos

🏆 Dicas de Ouro

  • 🔍 SEMPRE inspecione o código fonte (F12) - há pistas escondidas
  • 📝 Leia a descrição do puzzle com atenção - ela contém dicas
  • 🔄 Tente múltiplas abordagens - a primeira pode não funcionar
  • 📚 Use a documentação - os tutoriais foram feitos para ajudar
  • 🤝 Pergunte na comunidade - ninguém sabe tudo sozinho
  • ⏸️ Faça pausas - às vezes a solução vem quando você descansa

📋 Checklist do Iniciante

Li o guia "Como Começar"
