# ❓ Perguntas Frequentes (FAQ)

## 🎯 Sobre a Plataforma

### O que é o Não Desista Dev?
É uma plataforma educacional de investigação digital com 50 puzzles progressivos para aprender esteganografia, criptografia e forense.

### Preciso pagar para usar?
Não! A plataforma é completamente gratuita e de código aberto.

### Como meu progresso é salvo?
O progresso é salvo localmente no seu navegador (localStorage). Se você limpar os dados do navegador, seu progresso será perdido.

## 🎮 Sobre os Puzzles

### Como começar?
Recomendamos começar pelos puzzles 01-10 (Nível Iniciante) e avançar gradualmente.

### O que é uma flag?
Uma flag é uma string no formato `flag{...}` que comprova que você resolveu o puzzle.

### E se eu travar em um puzzle?
- Use as dicas disponíveis na página
- Consulte os tutoriais
- Participe do fórum da comunidade ( pesquise em outros foruns)
- Tente outra abordagem

### As flags são case-sensitive?
Sim! `flag{exemplo}` é diferente de `FLAG{EXEMPLO}`.

## 🛠️ Sobre as Ferramentas

### Preciso instalar algo?
A maioria das ferramentas online funciona direto no navegador. Para ferramentas mais avançadas, recomendamos usar as máquinas virtuais dos laboratórios.

### Onde encontro as ferramentas?
Na pasta `/tools` você encontra versões online simplificadas. Na pasta `/labs` há scripts para instalar as ferramentas completas.

## 💻 Sobre o Terminal Interativo

### O terminal é real?
É uma simulação para aprendizado. Os comandos executam funções pré-programadas que simulam um terminal Linux real.

### Quais comandos posso usar?
Digite `help` no terminal para ver a lista completa de comandos disponíveis.

## 🌐 Sobre os Laboratórios

### Como acesso os laboratórios?
Os laboratórios são ambientes virtuais que você pode rodar localmente com Vagrant ou Docker.

### Preciso de muito espaço em disco?
Cada laboratório pode ocupar de 2GB a 10GB dependendo da configuração.

## 📚 Sobre a Documentação

### Onde encontro tutoriais específicos?
Na pasta `/docs/tutoriais` você encontra guias detalhados sobre cada técnica.

### Existe um glossário?
Sim! Em `/docs/guias/glossary.md` você encontra todos os termos técnicos explicados.

## 🏆 Sobre Conquistas

### Como ganho conquistas?
Resolvendo puzzles! Algumas conquistas são especiais por completar todos os puzzles de um nível.

### Onde vejo meu ranking?
Na página da comunidade (`/community/rankings`), em breve.

## 🐛 Encontrei um bug

### Como reportar?
Abra uma issue no GitHub com:
- Descrição do problema
- Passos para reproduzir
- Screenshots (se aplicável)
- Navegador e sistema operacional

## 💡 Dicas Gerais

1. **Sempre inspecione o código fonte** (F12)

3. **Use as ferramentas recomendadas**
