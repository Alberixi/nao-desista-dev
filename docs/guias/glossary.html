> 📚 GLOSSÁRIO DO INVESTIGADOR

🔐 Criptografia

AES
Advanced Encryption Standard - Algoritmo de criptografia simétrica usado globalmente.
criptografia
Base64
Codificação que representa dados binários em texto ASCII (A-Z, a-z, 0-9, +, /).
codificação
Cifra de César
Substituição onde cada letra é deslocada um número fixo de posições no alfabeto.
clássica
Hash
Função unidirecional que gera uma impressão digital única dos dados.
criptografia
MD5
Algoritmo de hash de 128 bits (atualmente considerado inseguro para criptografia).
hash
RSA
Criptografia assimétrica baseada na fatoração de números primos grandes.
assimétrica
ROT13
Cifra de César com deslocamento de 13 posições (aplicar duas vezes volta ao original).
clássica
SHA
Secure Hash Algorithm - Família de funções hash (SHA-1, SHA-256, SHA-512).
hash
XOR
Operação lógica "ou exclusivo" usada em criptografia (reversível com a mesma chave).
operação

🕵️ Esteganografia

Esteganografia
Técnica de esconder informações dentro de outros arquivos (imagens, áudios, vídeos).
ocultação
LSB
Least Significant Bit - Bit menos significativo, usado para esconder dados em imagens/áudio.
imagem
Metadados
Dados sobre dados (EXIF em imagens, propriedades de arquivos, informações de documento).
informação
EXIF
Exchangeable Image File Format - Metadados em imagens (câmera, GPS, data).
imagem
Espectrograma
Representação visual do espectro de frequências de um áudio ao longo do tempo.
áudio
Carving
Técnica de recuperar arquivos baseada em suas assinaturas (magic numbers).
forense

🔍 Forense Digital

Dump de Memória
Cópia do conteúdo da RAM para análise forense (arquivos .raw, .mem).
memória
Imageamento
Cópia bit-a-bit de um disco ou partição para análise forense.
disco
IoC
Indicator of Compromise - Evidência de invasão ou atividade maliciosa.
indicador
PCAP
Formato de arquivo para captura de pacotes de rede (Packet CAPture).
rede
Volatility
Framework open source para análise de memória RAM.
ferramenta
Wireshark
Ferramenta de análise de protocolos de rede (sniffer de pacotes).
ferramenta

🌐 Rede

DNS
Domain Name System - Traduz nomes de domínio para endereços IP.
protocolo
HTTP
Hypertext Transfer Protocol - Protocolo de transferência de hipertexto.
protocolo
HTTPS
Versão segura do HTTP com criptografia SSL/TLS.
protocolo
IP
Internet Protocol - Endereço numérico que identifica um dispositivo na rede.
endereço
Porta
Ponto de comunicação em um dispositivo (ex: 80 para HTTP, 443 para HTTPS).
rede
TCP
Transmission Control Protocol - Protocolo confiável e orientado à conexão.
protocolo
UDP
User Datagram Protocol - Protocolo não confiável e sem conexão.
protocolo

🛡️ Segurança Web

CSRF
Cross-Site Request Forgery - Ataque que força usuário a executar ações indesejadas.
ataque
IDOR
Insecure Direct Object Reference - Acesso direto a objetos sem autorização.
vulnerabilidade
JWT
JSON Web Token - Token de autenticação em formato JSON assinado.
token
OAuth
Protocolo de autorização para acesso delegado a recursos.
protocolo
SQLi
SQL Injection - Injeção de comandos SQL maliciosos.
ataque
XSS
Cross-Site Scripting - Injeção de scripts maliciosos em páginas web.
ataque

💻 Comandos Linux

cat
Mostra conteúdo de arquivo (concatenate).
comando
chmod
Altera permissões de arquivos/diretórios.
comando
grep
Busca texto em arquivos (Global Regular Expression Print).
comando
ls
Lista arquivos e pastas (list).
comando
strings
Extrai texto legível de arquivos binários.
comando
xxd
Visualização hexadecimal de arquivos.
comando

📁 Tipos de Arquivo

.pcap
Captura de pacotes de rede (Wireshark, tcpdump).
extensão
.raw / .mem
Dump de memória RAM.
extensão
.dd / .img
Imagem de disco.
extensão
.elf
Executável Linux (Executable and Linkable Format).
extensão
.exe
Executável Windows.
extensão

🎯 Flags e Conceitos

Flag
String no formato flag{...} que comprova a solução do puzzle.
objetivo
CTF
Capture The Flag - Competição de segurança onde participantes buscam flags.
competição
OSINT
Open Source Intelligence - Inteligência de fontes abertas.
técnica
APT
Advanced Persistent Threat - Ameaça persistente avançada (ataques direcionados).
ameaça
Zero Day
Vulnerabilidade desconhecida pelo fabricante (0 dias de patch).
vulnerabilidade
