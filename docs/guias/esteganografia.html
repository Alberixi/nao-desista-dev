O que é Esteganografia?

Esteganografia é a técnica de esconder informações dentro de outros arquivos (imagens, áudios, vídeos) de forma que a existência da mensagem seja camuflada. Diferente da criptografia, que torna a mensagem ilegível, a esteganografia a torna invisível.

📌 Conceitos Básicos

1. LSB (Least Significant Bit)

O método mais comum em imagens. Altera o último bit de cada pixel para armazenar dados.

Pixel original: 10101100 10110011 10101101 Dado a esconder: 1 0 1 Pixel modificado: 10101101 10110010 10101101

Explicação: O último bit de cada cor foi alterado para armazenar os bits da mensagem. Visualmente, a diferença é imperceptível.

2. Metadados (EXIF)

Dados podem ser escondidos nos metadados de imagens (autor, câmera, GPS, etc.)

exiftool imagem.jpg # Resultado: # Author : flag{metadados_expostos} # GPS Position : 40.7128° N, 74.0060° W

3. Esteganografia em Áudio

  • Espectrograma: Mensagens visíveis nas frequências
  • LSB em áudio: Bits menos significativos das amostras
  • Eco: Informação codificada em ecos

🛠️ Ferramentas

Para Imagens

# Steghide steghide embed -cf imagem.jpg -ef mensagem.txt -pf senha steghide extract -sf imagem.jpg -pf senha # zsteg (PNG/BMP) zsteg imagem.png # Stegsolve (análise visual) java -jar stegsolve.jar # Exiftool (metadados) exiftool imagem.jpg # Binwalk (arquivos embutidos) binwalk imagem.png

Para Áudio

# Sonic Visualiser (espectrograma) sonic-visualiser audio.wav # Audacity (análise e LSB) audacity audio.wav # MP3Stego mp3stego -X -P senha audio.mp3

Para PDFs e Documentos

# pdfdetach (extrair anexos) pdfdetach -saveall documento.pdf # strings (texto legível) strings documento.pdf | grep flag # pdf-parser (análise de streams) pdf-parser documento.pdf

🔍 Técnicas de Detecção

1. Análise Visual

  • Inspecionar planos de bits em imagens (Stegsolve)
  • Verificar diferenças sutis nas cores

2. Análise Estatística

  • Testes qui-quadrado para LSB
  • Análise de entropia

3. Ferramentas Automáticas

# stegdetect stegdetect imagem.jpg # stegbreak stegbreak -r rules.ini -f imagem.jpg

🎯 Exemplos Práticos

Exemplo 1: Esconder texto em imagem com steghide

# Criar mensagem echo "flag{esteganografia_basica}" > mensagem.txt # Esconder na imagem steghide embed -cf gato.jpg -ef mensagem.txt -pf "" # Extrair depois steghide extract -sf gato.jpg -pf ""

Exemplo 2: LSB manual com Python

from PIL import Image def esconder_mensagem(imagem, mensagem): img = Image.open(imagem) pixels = list(img.getdata()) # Converter mensagem para binário bin_mensagem = ''.join(format(ord(c), '08b') for c in mensagem) # Esconder nos LSBs novos_pixels = [] for i, pixel in enumerate(pixels): if i < len(bin_mensagem): # Modificar LSB do vermelho novo_pixel=(pixel[0] & 0xFE) | int(bin_mensagem[i]) novos_pixels.append((novo_pixel, pixel[1], pixel[2])) else: novos_pixels.append(pixel) # Salvar nova imagem img.putdata(novos_pixels) img.save('imagem_com_segredo.png')

📚 Guia Passo a Passo

Para puzzles de imagem:

  1. Verifique metadados: exiftool imagem.jpg
  2. Analise visualmente: Stegsolve
  3. Teste ferramentas comuns: steghide, zsteg
  4. Varredura binária: binwalk imagem.jpg
  5. Extraia LSB: Scripts Python ou ferramentas dedicadas

Para puzzles de áudio:

  1. Escute: Pode ter mensagem em morse
  2. Espectrograma: Audacity/Sonic Visualiser
  3. LSB de áudio: Python com numpy
  4. Análise de eco: Ferramentas especializadas

Para PDFs/Documentos:

  1. strings: strings documento.pdf | grep flag
  2. Anexos: pdfdetach -saveall documento.pdf
  3. Metadados: exiftool documento.pdf
  4. Análise de streams: pdf-parser documento.pdf
💡 Dica do Mestre: Nem sempre a primeira camada é a resposta. Às vezes, a flag está escondida em múltiplas camadas de esteganografia.
⚠️ Atenção: Sempre faça backup dos arquivos originais antes de tentar extrair dados. Algumas ferramentas podem modificar o arquivo.
