Protocolos Principais

TCP (Transmission Control Protocol)

  • Orientado à conexão
  • Confiável (ACK, retransmissão)
  • Portas: 20,21(FTP), 22(SSH), 25(SMTP), 80(HTTP), 443(HTTPS)

UDP (User Datagram Protocol)

  • Não orientado à conexão
  • Não confiável (sem ACK)
  • Portas: 53(DNS), 67/68(DHCP), 69(TFTP), 123(NTP)

🛠️ Ferramentas de Análise

Wireshark

# Filtros comuns ip.src == 192.168.1.100 ip.dst == 8.8.8.8 tcp.port == 80 http.request.method == "GET" tcp contains "flag"

tshark

# Listar interfaces tshark -D # Analisar arquivo tshark -r captura.pcap # Filtrar e mostrar campos tshark -r captura.pcap -Y "http" -T fields -e ip.src -e ip.dst

tcpdump

# Capturar pacotes tcpdump -i eth0 -c 100 -w captura.pcap # Filtrar por porta tcpdump -i eth0 port 80 # Ler arquivo tcpdump -r captura.pcap -X
