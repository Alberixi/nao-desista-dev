> 🖼️ TUTORIAL DE ESTEGANOGRAFIA

O que é Esteganografia?

Esteganografia é a técnica de esconder informações dentro de outros arquivos (imagens, áudios, vídeos) de forma que a existência da mensagem seja camuflada. Diferente da criptografia, que torna a mensagem ilegível, a esteganografia a torna invisível.

📌 Conceitos Básicos

1. LSB (Least Significant Bit)

O método mais comum em imagens. Altera o último bit de cada pixel para armazenar dados.

Pixel original: 10101100 10110011 10101101 Dado a esconder: 1 0 1 Pixel modificado: 10101101 10110010 10101101

2. Metadados (EXIF)

Dados podem ser escondidos nos metadados de imagens (autor, câmera, GPS, etc.)

exiftool imagem.jpg # Resultado: # Author : flag{metadados_expostos} # GPS Position : 40.7128° N, 74.0060° W

🛠️ Ferramentas

Para Imagens

# Steghide steghide embed -cf imagem.jpg -ef mensagem.txt -pf senha steghide extract -sf imagem.jpg -pf senha # zsteg (PNG/BMP) zsteg imagem.png # Exiftool (metadados) exiftool imagem.jpg # Binwalk (arquivos embutidos) binwalk imagem.png

🧪 Exemplos Práticos

Exemplo 1: Esconder texto em imagem com steghide

# Criar mensagem echo "flag{esteganografia_basica}" > mensagem.txt # Esconder na imagem steghide embed -cf gato.jpg -ef mensagem.txt -pf "" # Extrair depois steghide extract -sf gato.jpg -pf ""

Exemplo 2: LSB com Python

from PIL import Image def esconder_mensagem(imagem, mensagem): img = Image.open(imagem) pixels = list(img.getdata()) # Converter mensagem para binário bin_mensagem = ''.join(format(ord(c), '08b') for c in mensagem) # Esconder nos LSBs novos_pixels = [] for i, pixel in enumerate(pixels): if i < len(bin_mensagem): novo_pixel=(pixel[0] & 0xFE) | int(bin_mensagem[i]) novos_pixels.append((novo_pixel, pixel[1], pixel[2])) else: novos_pixels.append(pixel) img.putdata(novos_pixels) img.save('imagem_com_segredo.png')
💡 Dica: Sempre verifique metadados (exiftool) primeiro, é a técnica mais simples!
