## 📋 Sobre o Projeto

O **Não Desista Dev** é uma plataforma educacional que oferece desafios progressivos de investigação digital. Cada puzzle é projetado para ensinar conceitos fundamentais de segurança da informação de maneira prática e envolvente.

### 🎯 Objetivos
- Ensinar técnicas de esteganografia
- Introduzir conceitos de criptografia
- Desenvolver habilidades forenses
- Praticar comandos Linux
- Estimular o raciocínio lógico

---

## 🚀 Começando

### Pré-requisitos
- Navegador moderno (Chrome, Firefox, Edge)
- Conhecimentos básicos de terminal (opcional)
- Curiosidade e persistência!
- **Sempre verificar links no VirusTotal**

### Instalação

```
# Clone o repositório
git clone https://github.com/seu-usuario/nao-desista-dev.git

# Entre na pasta
cd nao-desista-dev

# Inicie um servidor local (opcional)
npx live-server
