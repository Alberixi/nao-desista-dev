> 🤝 GUIA DE CONTRIBUIÇÃO

Como ajudar a construir o Não Desista Dev

💡 Primeiro passo: Leia este guia completo antes de começar.

📋 Formas de Contribuir

🐛 Reportar Bugs 💡 Sugerir Puzzles 📝 Documentação 🔧 Código 🌐 Traduções 🎨 Design 📢 Divulgação

🐛 Reportando Bugs

1
Verifique se o bug já foi reportado
2
Crie uma issue no GitHub
Título: [BUG] Descrição clara Corpo: - Descrição do problema - Passos para reproduzir - Comportamento esperado - Screenshots (se aplicável) - Ambiente (SO, navegador)

💡 Sugerindo Novos Puzzles

1
Escolha um tema e nível
2
Crie o puzzle em HTML
<!-- Estrutura básica --> <!DOCTYPE html> <html> <head> <title>Novo Puzzle</title> <link rel="stylesheet" href="../../assets/css/style.css"> </head> <body> <!-- Seu puzzle aqui --> </body> </html>
3
Envie via Pull Request

📝 Contribuindo com Documentação

A documentação fica em /docs/. Estrutura:

docs/ ├── guias/ # Guias práticos ├── tutoriais/ # Tutoriais completos └── referencias/ # Referências técnicas

🔧 Contribuindo com Código

Setup do Ambiente

# Clone o repositório git clone https://github.com/seu-usuario/nao-desista-dev.git cd nao-desista-dev # Instale dependências (opcional) npm install # Inicie servidor local npx live-server

Padrões de Código

// JavaScript: Use const/let, arrow functions // CSS: Use variáveis CSS (--bg-primary, --text-primary) // HTML: Indentação com 2 espaços

Commit Messages

feat: Adiciona novo puzzle X fix: Corrige erro no terminal interativo docs: Atualiza documentação de esteganografia style: Melhora layout da página inicial refactor: Simplifica função processCommand

🎨 Diretrizes de Design

  • Tema escuro com verde (#0f0) como cor principal
  • Usar a classe .terminal-header para cabeçalhos
  • Mensagens de erro em vermelho (#f00)
  • Dicas em azul (#66c0f0)

📊 Processo de Pull Request

① Fork o repositório
② Crie uma branch: git checkout -b minha-feature
③ Commit suas mudanças
④ Push para a branch: git push origin minha-feature
⑤ Abra um Pull Request no GitHub

🏅 Reconhecimento

Contribuidores serão adicionados:

  • No Hall da Fama da comunidade
  • Com badges especiais no perfil
  • Menção nos releases
❓ Dúvidas? Abra uma issue ou pergunte no Discord!
